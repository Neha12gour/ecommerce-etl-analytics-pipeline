# 🛒 E-Commerce ETL Analytics Pipeline & Customer Insights Dashboard

## 📌 Project Overview

This project demonstrates an end-to-end Data Engineering and Analytics workflow using real-world E-Commerce data.

The project extracts data from multiple API endpoints, performs data cleaning and transformation, stores the processed data in SQL, and visualizes business insights through an interactive Power BI dashboard.

The objective is to build a scalable data pipeline and generate actionable insights for business decision-making.

## 📂 Dataset Information

The project uses data extracted from 3 API endpoints:

### 1. Users Data
Contains:
- Customer Information
- Gender
- City
- Customer Details

### 2. Products Data
Contains:
- Product Information
- Categories
- Pricing Data
- Product Ratings

### 3. Carts Data
Contains:
- Orders
- Quantity Purchased
- Customer Purchases
- Revenue Information

---

## ⚙️ ETL Process

### Extract
- Retrieved data from 3 API endpoints.
- Converted JSON responses into structured datasets.

### Transform
- Removed duplicates.
- Handled missing values.
- Standardized data formats.
- Created analytical metrics.
- Prepared relational datasets for SQL storage.

### Load
- Loaded cleaned datasets into SQL tables.
- Created relationships between tables.
- Optimized data for reporting and analytics.
